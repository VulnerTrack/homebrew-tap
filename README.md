# VulnerTrack Homebrew Tap

Homebrew formulae for VulnerTrack tools.

## Installation

```bash
brew install --cask VulnerTrack/tap/kite-collector
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| kite-collector | Cybersecurity asset discovery, configuration audit, and posture analysis agent |

## How it works

Formulae in `Formula/` are auto-generated and pushed by [GoReleaser](https://goreleaser.com/) on every `v*` tag release of [kite-collector](https://github.com/VulnerTrack/kite-collector).
